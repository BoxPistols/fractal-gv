## Get Start

``` 
git clone https://github.com/BoxPistols/fractal-gv.git
```

### first setting \(once\)

``` 
yarn run install
```

### start move

``` 
yarn run dev
```

### sass compile

* sass to css

``` 
yarn run css
```

### build

* static files
* compile to docs/

``` 
yarn run build
```
